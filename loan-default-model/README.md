# Loan Default Prediction

## 📌 Problem
The goal of this project is to predict whether a customer will default on a loan using financial data.

## 🧠 Approach
- Logistic Regression
- Features: income, fico score, balance, inquiries

## 📊 Results
- Evaluated using accuracy and confusion matrix

## ⚠️ Insights
- High fico score → lower default risk
- Dataset may have bias (only approved loans)

## 🛠 Tools
Python, Pandas, Scikit-learn, Statsmodels
