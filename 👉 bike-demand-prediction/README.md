## 🚴 Bike Demand Prediction

### 📌 Problem
The goal of this project is to predict bike rental demand based on weather and seasonal factors.

---

### 📊 Dataset
The dataset includes:
- Temperature
- Humidity
- Windspeed
- Season
- Weather conditions
- Date-related features
- Bike rental count (target variable)

---

### 🔍 Approach
- Performed data cleaning and preprocessing  
- Conducted exploratory data analysis (EDA)  
- Engineered features such as season and weekday  
- Built a Linear Regression model  

---

### 📈 Results
- Identified key factors affecting demand such as temperature and season  
- Observed reduced demand during poor weather conditions  
- Model performance evaluated using R² and RMSE  

---

### 💼 Business Impact
This model can help bike-sharing companies:
- Optimize bike availability  
- Improve resource planning  
- Enhance customer satisfaction
